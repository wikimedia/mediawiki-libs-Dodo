# dodo-test-generator
Test generator for Dodo library

## How to install it
```bash
$ composer install
$ npm install
```

## How to run
```bash
$ composer robo build
```

Arguments:
* rewrite => false
* limit => -1
* phpcbf => false
* run => true
* cleanup => false
* compact => false
